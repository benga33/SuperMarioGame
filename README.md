# Mario Escape From the Princess

## Opis

Mario Escape From the Princess je 2D mobilna igra napravljena u MIT App Inventoru. Igrač upravlja Mariom i izbjegava prepreke koje dolaze prema njemu. Cilj je preživjeti što duže i ostvariti što veći score.

## Funkcionalnosti

* start igre
* kontrola igrača (tap za skok)
* prepreke koje se kreću
* detekcija sudara
* game over ekran
* restart igre
* score sistem

## Tehnologija

* MIT App Inventor

## Struktura aplikacije

### Screen1

* Canvas za prikaz igre
* ImageSprite (Mario)
* ImageSprite (prepreke)
* Label za score
* Button za start

### Logika igre

* Clock komponenta za kretanje objekata
* provjera sudara između sprite-ova
* povećavanje score-a kroz vrijeme

## Pokretanje aplikacije

1. Otvori MIT App Inventor
2. Klikni "Import project (.aia)"
3. Učitaj .aia fajl
4. Pokreni emulator ili poveži telefon
5. Klikni "Connect" → AI Companion
